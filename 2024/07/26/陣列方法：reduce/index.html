    陣列方法：reduce()

    前端 > JavaScript > 陣列 JavaScript 陣列 陣列方法
    發佈時間 : 2024-07-26 15:42
    字數:720 閱讀 :
    評論:
    1. 描述
    2. 如何使用
    3. 範例

    描述

    reduce() 方法會針每一個目前迭代到的陣列元素 ( 空值除外 )，去執行 callback function，而 callback function 會接受 4 個參數：

    • accumulator ：累加器 ( 必填 )
    • currentValue ：目前迭代到的元素 ( 必填 )
    • index ：目前迭代到的元素的索引 ( 選擇性 )
    • array ：陣列本身 ( 選擇性 )

    又其中 accumulator 與 currentValue 的初始值會有兩種不同的情況，呼叫 reduce() 時，有無提供 initialValue？（建議都要提供 initialValue，比較安全，可參考 MDN Array.prototype.reduce()）

    1. 有提供 initialValue，則 accumulator 一開始的值會等於 initialValue，currentValue 會等於陣列中的第一個元素值。

    2. 未提供 initialValue，則 accumulator 一開始的值會等於陣列中的第一個元素值，currentValue 會等於陣列中的第二個元素值。

    然後在每一次迭代中，callback function 執行完後都會吐一個值，此值會成為下一次迭代的 accumulator的值。

    當所有迭代都完成後，最後一次吐出來的 accumulator 的值，會是一個「加總」、「集合」等等。

    如何使用

    使用 reduce() 方法時可以帶入一個 initialValue 以及要傳入一個 callback function，該 function 可傳入四個參數：

    • accumulator ：累加器 ( 必填 )
    • currentValue ：目前迭代到的元素 ( 必填 )
    • index ：目前迭代到的元素的索引 ( 選擇性 )
    • array ：陣列本身 ( 選擇性 )
    // 傳統函式 ( 此例未提供 initialValue )
[0, 1, 2, 3, 4].reduce(
  function (accumulator, currentValue, currentIndex, array) {
    return accumulator + currentValue;
  },
);

// 箭頭函式 ( 此例有提供 initialValue， initialValue 值為 10)
[2, 4, 6, 8, 10].reduce((accumulator, currentValue, currentIndex, array) => {
  return accumulator + currentValue;
}, 10);

    將上述的箭頭函式的例子，攤開每一次迭代的狀況，就如下表格：

    callback 次數 accumulator currentValue currentIndex array return value
    first 10 2 0 [2, 4, 6, 8, 10] 10 + 2 = 12
    second 12 4 1 [2, 4, 6, 8, 10] 10 + 4 = 14
    third 14 6 2 [2, 4, 6, 8, 10] 14 + 6 = 20
    fourth 20 8 3 [2, 4, 6, 8, 10] 20 + 8 = 28
    fifth 28 10 4 [2, 4, 6, 8, 10] 28 + 10 = 38

    範例

    1. 加總所有陣列之元素值
    // 傳統函式
[0, 1, 2, 3, 4].reduce((accumulator, currentValue, currentIndex, array) => {
  return accumulator + currentValue;
}, 0);
// 0 + 0 + 1 + 2 + 3 + 4 = 10，sum = 10。

// 箭頭函式
[0, 1, 2, 3, 4].reduce((accumulator, currentValue, currentIndex, array) => {
  return accumulator + currentValue;
}, 0);
// 0 + 0 + 1 + 2 + 3 + 4 = 10，sum = 10。
    1. 計算陣列中每個元素的重複次數，並以物件的形式顯示
    // 傳統函式
let names = ["Alice", "Bob", "Tiff", "Bruce", "Alice"];

let countedNames = names.reduce(function (allNames, name) {
  if (name in allNames) {
    allNames[name]++;
  } else {
    allNames[name] = 1;
  }
  return allNames;
}, {});
// countedNames is: { 'Alice': 2, 'Bob': 1, 'Tiff': 1, 'Bruce': 1 }

// 箭頭函式
let names = ["Alice", "Bob", "Tiff", "Bruce", "Alice"];

let countedNames = names.reduce((allNames, name) => {
  if (name in allNames) {
    allNames[name]++;
  } else {
    allNames[name] = 1;
  }
  return allNames;
}, {});
// countedNames is: { 'Alice': 2, 'Bob': 1, 'Tiff': 1, 'Bruce': 1 }
