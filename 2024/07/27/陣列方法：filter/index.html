    陣列方法：filter()

    前端 > JavaScript > 陣列 JavaScript 陣列 陣列方法
    發佈時間 : 2024-07-27 14:17
    字數:340 閱讀 :
    評論:
    1. 描述
    2. 如何使用
    3. 範例

    描述

    filter() 方法會將陣列中的所有元素，依序地傳入一次至 callback function 中，並將回傳值為 Truthy 的元素建構成一個新的陣列，也就是篩選出符合條件的元素出來，並將這些元素塞進一個新的陣列裡。

    如何使用

    使用 filter() 方法時要傳入一個 callback function，該 function 可傳入三個參數：

    1. currentValue (必要) ：原陣列目前迭代挑到的元素。
    2. index (選擇性) ：原陣列目前迭代挑到的元素的索引。
    3. array (選擇性) ：原陣列本身。
    let newArr = arr.filter((currentValue, index, array) => {
  // do something
})

    範例

    1. 過濾小於 10 的數字。
    function isBigEnough(value) {
  return value >= 10;
}

let filtered = [12, 5, 8, 130, 44].filter(isBigEnough);
console.log(filtered); // 12, 130, 44
    1. 過濾 JSON 檔中無效的項目
      以下範例會用 filter() 把 id 屬性為非數值型別的項目過濾掉。
    var arr = [
  { id: 15 },
  { id: -1 },
  { id: 0 },
  { id: 3 },
  { id: 12.2 },
  {},
  { id: null },
  { id: NaN },
  { id: "undefined" },
];

function isNumber(obj) {
  return obj !== undefined && typeof obj === "number" && !isNaN(obj);
}

function filterByID(item) {
  if (isNumber(item.id)) {
    return true;
  }
  return false;
}

var arrByID = arr.filter(filterByID);
console.log("過濾好的陣列\n", arrByID);
// 過濾好的陣列
// [{ id: 15 }, { id: -1 }, { id: 0 }, { id: 3 }, { id: 12.2 }]
