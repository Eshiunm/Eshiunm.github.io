    變數與作用域 6：undefined 與 null 的差異

    基本概念

    在 Javascript 中有兩大型別，其中 undefined 與 null 都是屬於 Primitive value( 原始值或稱基本型別 )，
    而原始值是無法被更改的！，
    例如 Boolean 只會有 true 和 false 這兩種值，因此 null 、undefined 也是一樣。
    undefined 型別只有一種值，就是 undefined。
    null 型別只有一種值，就是 null。

    undefined 的定義

    undefined 表示一個變數已經被宣告，但是該變數的值尚未定義。
    『undefined means a variable has been declared but has not yet been assigned a value 。』
    例：

    let test;
console.log(test); //shows undefined
console.log(typeof test); //shows undefined

    null 的定義

    null 是一個值，可以被賦予到一個變數上，來表示該變數目前的值是空值。
    『null is an assignment value. It can be assigned to a variable as a representation of no value』

    let test = null;
console.log(test); //shows undefined
console.log(typeof test); //shows object
/*
  注意：
  變數目前的值是空值不等於變數目前的值為 0，
  空值 ≠ 0。
*/

    從定義了解差異

    從前面兩個程式碼的例子中可以發現，undefined 和 null 是兩種不同的型別

    console.log(null === undefined) // false (not the same type)
console.log(null == undefined) // true (but the "same value")
console.log(null === null) // true (both type and value are the same)
    undefined null
    定義 表示某變數已經被宣告，但該變數的值尚未定義 變數的值已經定義，值是 null，表示變數目前的值是空值

    使用情境來理解差異

    • name is undefined
    console.log(name); // name is undefined
var name;

    You: What is name?
    JavaScript: name? What’s a name? I don’t know what you’re talking about. You haven’t ever mentioned any name before. Are you seeing some other scripting language on the client-side?

    • name = null
    let name = null;

    You: What is name?
    JavaScript: I don’t

    • name = false
    let name = false;

    You: What is name?
    JavaScript: Boolean false.

    • name = “”
    let name = "";

    You: What is name?
    JavaScript: Empty string.

    一張圖理解差異

    undefined 可以當作變數名稱或參數使用

    驚不驚喜，意不意外～我們常說不能拿保留字當作變數，沒想到 undefined 居然打破這個限制！！！在非全域作用範圍下， undefined 允許被當成變數名稱，而且變數的值是可以被修改的。

    // 當作變數名稱
(function() {
  var undefined = 'foo';
  console.log(`值為${undefined}，型別為${typeof undefined}`); // 顯示：值為foo，型別為string
})()

// 當作參數
(function(undefined) {
  console.log(`值為${undefined}，型別為${typeof undefined}`); // 顯示：值為foo，型別為string
})('foo');
